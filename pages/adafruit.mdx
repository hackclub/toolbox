# Adafruit for Hack Clubbers

Adafruit Industries is an open-source hardware company based in New York City. It designs, manufactures and sells a number of electronics products, electronics components, tools and accessories. If you are starting a electronics project it's a great place to visit. 

**All Hack Clubbers can receive a 10% discount using the code `HACKCLUB`.**
