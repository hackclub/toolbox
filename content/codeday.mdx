# CodeDay for Hack Clubbers

CodeDay is a worldwide event where student programmers, artists, musicians, actors, and everyone else can get together and build apps & games for 24 hours. Students of all skill levels are welcome — there will be workshops and mentors who can help if you're new! 

This season, CodeDay is offering a 234872394% discount for Hack Clubbers to attend in-person CodeDay events across the US. Use the code `HACKCLUB` when you register for any of the following in-person events:

* [Pittsburgh](https://event.codeday.org/pittsburgh)
* [Atlanta](https://event.codeday.org/atlanta)
* [Washington DC](https://event.codeday.org/dc)
* [Baltimore](https://event.codeday.org/baltimore)
* [Phoenix](https://event.codeday.org/phoenix)
* [Bay Area](https://event.codeday.org/ba)

Don't live near an in-person event. Join [virtually](https://virtual.codeday.org) at no cost!
